Проверка розыгрыша Claude

Независимый verifier для розыгрыша. Убеждается что организатор не подменил seed и не изменил базу участников. Алгоритм детерминированный — из одного seed'а всегда получаются одни и те же победители.

📋 Как проверить

  1. Получи у организатора proof.json (в ТГ-канале после розыгрыша)
  2. Вставь содержимое в поле ниже
  3. Нажми "Проверить" — скрипт прогонит алгоритм заново
  4. Если 3 галочки зелёные — розыгрыш честный
  5. Можешь также проверить участвовал ли ты лично (без раскрытия email)

🔬 Алгоритм (для разработчиков)

Всё воспроизводимо в любом языке — SHA-256 из стандартной библиотеки:

  1. Участники сортируются по email (lowercase, ASC). Каждый = {email_hash: SHA-256(email), weight: 1 + кол-во_бонусов}
  2. Для каждого раунда N (начиная с 0):
    • h = SHA-256(seed + ":" + N)
    • target = int(h[0..16], 16) mod total_weight
  3. Идём по отсортированному списку, накапливаем cumulative += weight, победитель = первый где target < cumulative
  4. Удаляем победителя из пула, переходим к следующему раунду

Первые 5 раундов — Claude Pro, шестой (последний) — Claude Code Max.
